## STRUCTURE GÉNÉRALE:

### Cours 1 – 3 : Intro/Recap HTML/CSS  
Aperçu sur les outils du cours + bases de HTML/CSS.  

### Cours 3 – 7 : Introduction à p5.js et JavaScript  
Bases de la programmation (créative) en JavaScript à travers la bibliothèque p5.js.  

### Cours 8 – 10 : Laboratoire  
Vous apprendrez à intégrer vos sketchs p5.js dans un site et à utiliser GitHub pour publier gratuitement vos projets en ligne.
